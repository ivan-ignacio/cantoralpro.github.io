<p>Esta política de privacidad rige el uso de la aplicación de software <strong>Cantoral Pro</strong> para dispositivos móviles, creada por <strong>Ivan (Desarrollo Independiente)</strong>.</p>

<h2>1. Información que la Aplicación Obtiene</h2>
<p><strong>Información proporcionada por el usuario:</strong> La aplicación requiere un inicio de sesión exclusivo para administradores utilizando correo electrónico y contraseña a través de los servicios de Firebase Authentication. No se requiere registro para usuarios que únicamente consultan el cancionero.</p>
<p><strong>Recopilación automática:</strong> La aplicación puede recopilar cierta información de forma automática, incluyendo, entre otros, el tipo de dispositivo móvil que utiliza, el sistema operativo de su dispositivo y la forma en que utiliza la aplicación para fines de diagnóstico y mejora del servicio.</p>

<h2>2. Uso de Firebase y Servicios de Terceros</h2>
<p>Esta aplicación utiliza servicios de terceros que pueden recopilar información utilizada para identificarlo:</p>
<ul>
    <li><strong>Google Play Services:</strong> Para el funcionamiento básico en Android.</li>
    <li><strong>Firebase Realtime Database:</strong> Para el almacenamiento y sincronización de las letras y acordes de las canciones.</li>
    <li><strong>Firebase Authentication:</strong> Para la gestión segura del acceso de administradores.</li>
</ul>

<h2>3. Privacidad de los Niños</h2>
<p>Nuestros servicios no se dirigen a personas menores de 13 años. No recopilamos a sabiendas información de identificación personal de niños menores de 13 años. Si descubrimos que un niño menor de 13 años nos ha proporcionado información personal, la eliminaremos inmediatamente de nuestros servidores.</p>

<h2>4. Seguridad</h2>
<p>Nos preocupamos por salvaguardar la confidencialidad de su información. Proporcionamos salvaguardas físicas, electrónicas y de procedimiento para proteger la información que procesamos y mantenemos, limitando el acceso a personal autorizado.</p>

<h2>5. Cambios</h2>
<p>Esta Política de Privacidad puede ser actualizada de vez en cuando por cualquier razón. Se le notificará de cualquier cambio en nuestra Política de Privacidad mediante la publicación de la nueva política aquí.</p>
